# Site Cálculo de Média

Projeto simples desenvolvido para calcular a média de três notas informadas pelo usuário.

O usuário digita as três notas nos campos disponíveis e o sistema calcula automaticamente a média final, mostrando o resultado na tela.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript

## Funcionalidades

* Inserção de três notas
* Cálculo automático da média
* Exibição do resultado para o usuário

## Objetivo do projeto

Este projeto foi criado com o objetivo de praticar conceitos básicos de desenvolvimento front-end, como estruturação com HTML, estilização com CSS e lógica de programação utilizando JavaScript.

## Como usar

1. Digite as três notas nos campos disponíveis.
2. Clique no botão para calcular a média.
3. O resultado será exibido na tela.
